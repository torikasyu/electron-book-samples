## 開発環境

- Node.js (最新のLTS版を推奨)
- npm または yarn
- Visual Studio Code (推奨)

## 各アプリケーションの実行方法

各ディレクトリに移動して、以下のコマンドを実行してください：

```bash
# 依存パッケージのインストール
npm install

# 開発モードでアプリケーションを実行
npm run dev
```

## 技術スタック

- **Electron**: デスクトップアプリケーションフレームワーク
- **React**: UIライブラリ
- **TypeScript**: 静的型付けJavaScript
- **Supabase Storage**: 画像ストレージ (chapter06)

## アーキテクチャ

このサンプルコードでは、関数ベースの設計とクリーンアーキテクチャを採用しています。特にchapter06では、以下のような層に分けた設計を実践しています：

1. **ドメイン層**: ビジネスロジックの中核となるデータ構造とルール
2. **ユースケース層**: アプリケーション固有のビジネスルール
3. **インターフェース層**: UIやデータベースなどの外部要素との接続
4. **インフラストラクチャ層**: フレームワークやライブラリなどの外部ツール
