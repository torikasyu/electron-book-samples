# 第3章: 基本的なアプリケーション開発

このディレクトリには、Electron入門書の第3章で解説されている基本的なアプリケーション開発のサンプルコードが含まれています。

## 内容

- Hello Worldアプリの作成
- メインプロセスとレンダラープロセスの理解
- IPCを使ったプロセス間通信

## プロジェクト構造

```
chapter04/
├── src/
│   ├── main/              # メインプロセス関連のコード
│   │   └── main.ts        # メインプロセスのエントリーポイント
│   ├── renderer/          # レンダラープロセス関連のコード
│   │   ├── App.tsx        # メインのReactコンポーネント
│   │   ├── index.html     # HTMLテンプレート
│   │   └── index.tsx      # レンダラープロセスのエントリーポイント
│   └── preload/           # プリロードスクリプト
│       └── preload.ts     # プリロードスクリプト
├── package.json           # プロジェクト設定
├── tsconfig.json          # TypeScript設定
└── webpack.config.js      # Webpack設定
```

## 実行方法

```bash
# 依存パッケージのインストール
npm install

# 開発モードでアプリケーションを実行
npm run dev
```

## 学習ポイント

1. **メインプロセスとレンダラープロセス**
   - Electronアプリケーションの2つの実行コンテキスト
   - それぞれの役割と違い

2. **プリロードスクリプト**
   - セキュリティ上の理由からのコンテキスト分離
   - 安全なブリッジの作成方法

3. **IPC通信**
   - プロセス間でのメッセージのやり取り
   - 非同期通信と同期通信の違い
